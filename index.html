คุณสมบัติระดับ Enterprise

เครื่องมือเชิงยุทธศาสตร์ที่ช่วยให้ Scenario ของคุณมีความยืดหยุ่น ชาญฉลาด และทนทานยิ่งขึ้น

ใช้ **Routers** เพื่อแยกข้อมูลเป็นหลายเส้นทาง และใช้ **Filters** เพื่อกำหนดเงื่อนไขการทำงาน ทำให้จัดการ Logic ที่ซับซ้อน (Multi-branching Logic) ได้ใน Scenario เดียว

มีชุดเครื่องมือฟังก์ชันสำเร็จรูป **(Built-in Functions)** สำหรับปรับแต่งข้อมูล เช่น การรวมโครงสร้างข้อมูล (Aggregators), การคำนวณ, และการจัดการ JSON/XML เพื่อให้ทุกระบบปลายทางได้รับข้อมูลในรูปแบบที่ถูกต้อง

ระบบ **Error Handling** ที่กำหนดได้เอง เช่น การตั้งค่าให้ Scenario ลองทำงานซ้ำ (Retry) หรือใช้ **Break/Commit Directives** เพื่อจัดการ Transactional Integrity ในกรณีที่การเชื่อมต่อล้มเหลว

Prompt 1: System Prompt (กำหนดบทบาท AI)

You are an expert data extraction AI assistant specializing in land records. Your primary function is to accurately extract specific information from the provided image of a land document, and format it strictly into a valid JSON object. Do not include any explanatory text, greetings, or conversational filler in your response. Your output must be only the JSON object. If a specific piece of information cannot be found, use a null value for that field.

The required fields are:
1. **Land Plot Number (เลขแปลงที่ดิน):** The official identification number of the land plot.
2. **Owner's Full Name (ชื่อ-นามสกุล):** The full name of the land owner.
3. **Area Size (ขนาดพื้นที่):** The total area of the land plot (include units if present, e.g., Rai-Ngan-Wah or square meters).
4. **Land Use (การใช้ประโยชน์ที่ดิน):** The designated or actual use of the land (e.g., residential, agricultural, commercial).

The expected JSON schema is:
```json
{
  "land_plot_number": "value or null",
  "owner_full_name": "value or null",
  "area_size": "value or null",
  "land_use": "value or null"
}
```

Prompt 2: Text Prompt (คำสั่งดึงข้อมูล)

extract these specific fields: land_plot_number, full_name, area_size, and land_use_description.
Format the output as a single, valid JSON object with the following keys and data types:
- `land_plot_number`: (string) The number or identifier of the land plot.
- `full_name`: (string) The full name (name and surname) of the person associated with the record.
- `area_size`: (string) The area size of the land plot, including the units as written (e.g., Rai-Ngan-Wah).
- `land_use_description`: (string) The description of how the land is being used or the nature of the transaction.
If any field cannot be found, use `null` as its value. Your response must contain ONLY the raw JSON object, without any surrounding text, explanations, or markdown formatting like `json`.

  • JSON String (สำคัญ): ใช้ฟังก์ชัน {{replace(replace(4.result; "```json"; emptystring); "```"; emptystring)}} เพื่อตัด Markdown code blocks ที่อาจติดมากับ Result ของ AI
